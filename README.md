# Пример README.md для лабораторной работы

## 1. Форматирование текста
В этом разделе демонстрируются различные способы форматирования текста:
- *Текст курсивом*
- **Полужирный текст**
- ***Полужирный курсив***
- ~~Зачеркнутый текст~~
- `Моноширинный текст для кода`

## 2. Списки
### Нумерованный список:
1. Первый пункт
2. Второй пункт
3. Третий пункт

### Маркированный список:
- Элемент A
- Элемент B
- Элемент C
